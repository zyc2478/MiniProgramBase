# MiniProgramBase

WeChat Mini Program base projects.

## Projects

### 1. SkyWalking-Catering (SkyWalking 西餐店)
A professional catering mini program designed for high-end restaurants.
- **Features**: Menu browsing, Online ordering, Membership center.
- **Tech**: WeChat Mini Program, WeUI.

### 2. SkyCampus (Sky校园)
A comprehensive campus service mini program for students.
- **Features**: Campus news, Service discovery (Schedule, Library, Bus), Student profile.
- **Tech**: WeChat Mini Program, WeUI.

## How to Run
1. Download and install [WeChat DevTools](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html).
2. Open WeChat DevTools and select "Import Project".
3. Choose either `SkyWalking-Catering` or `SkyCampus` folder.
4. The projects use `useExtendedLib: weui`, so no manual installation of WeUI is required.
